# PyAsteroids

A clone-inspired arcade shooter built in Python with pygame. Survive waves of incoming asteroids, rack up a score, and see how long you can last as the difficulty ramps up each level.

---

## Gameplay

Asteroids spawn from off-screen and fly toward you. Shoot them down before they hit your ship. Each level the asteroids get faster and bigger — how far can you get?

---

## Features

- Sprite-based player ship with rotation and directional movement
- Escalating difficulty — each level reduces spawn intervals and increases asteroid speed
- Score tracking with a final score screen on death
- Sound effects and ambient audio with pause support
- Animated asteroid destruction

---

## Controls

| Action | Input |
|---|---|
| Rotate Left | A |
| Rotate Right | D |
| Move Forward | W |
| Move Backward | S |
| Shoot | Space |
| Pause | Escape |

---

## Running the Game

**Requirements:**
- Python 3.x
- pygame

**Install pygame:**
```bash
pip install pygame
```

**Run:**
```bash
python Py-Asteroids.py
```
